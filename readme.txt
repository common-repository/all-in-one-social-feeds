== Description ==
This plugin helps to display latest feeds from facebook, twitter,instagram, pinterest and youtube with tabs using a widget.

<strong>Features</strong>

* Widgets based
* Facebook feed
* Twitter feed
* Instagram feed
* Pinterest feed
* Youtube feed
* Tab View
* Easy to customize

<strong>Required</strong>

* The PHP allow_url_fopen setting must be enabled for the plugin.

== Installation ==
1. Upload the folder "auto-login-after-registrationt" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Wp-admin-> All in one Social Feeds , put all settings .
4. Go to Wp-admin->Appearance -> Widgets , here is 'All in one Social Feeds' .
5. Put 'All in one Social Feeds' widget in widget area.
6. Call widget area in your theme.
